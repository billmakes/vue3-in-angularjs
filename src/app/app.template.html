Vue3, AngularJs, Events

A demonstration of Vue3 components inside an AngularJs component, with events, using a custom directive.

AngularJs App Component

  • {{$ctrl.data}}
  • {{$ctrl.greeting|json}}
  • {{$ctrl.message}}
  • {{scopeVar|json}}

'ng-vue' is an AngularJs directive. It loads a Vue component by finding a Vue component name from it's element name. 

  <FirstVueComponent ng-vue
    greeting="$ctrl.greeting"
    message="$ctrl.message"
    data="$ctrl.data"
    ng-click="$ctrl.alert($event)"
    v-on:my-event="$ctrl.handleEvent($event)"></MyVueComponent>
  
   <SecondVueComponent ng-vue greeting="$ctrl.greeting">
   </SecondVueComponent>
